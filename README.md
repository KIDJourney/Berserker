# Berserker

Web Application Smoking Test.

# requirement

- python 3.x
- gevent >= 1.1.2
- requests>=2.3.0

# install

   $ pip install berserker

# Usage

    usage: berserker [-h] [-c CONCURRENCY] [-n REQUESTS] [-m METHOD]
                     [-H CUSTOM_HEADER [CUSTOM_HEADER ...]]
                     [-C CUSTOM_COOKIE [CUSTOM_COOKIE ...]]
                     [url]
